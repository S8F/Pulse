
# Pulse Team Status Dashboard

This folder is the raw standalone Figma Make export for Pulse Team Status Dashboard. The original design is available at [Figma](https://www.figma.com/design/Hmmy9M8jjVj2NkdxalZ3jv/Pulse-Team-Status-Dashboard).

In this repository, the export is included as reference material under `src/figma-output/`. The assessment app you should actually work on lives at the repo root and is documented in the root [`README.md`](../../README.md).

## Running this standalone export

Only do this if you specifically want to inspect the generated bundle in isolation.

From `src/figma-output/`:

```bash
npm install
npm run dev
```

For the real assessment workflow, use the repo-root commands instead:

```bash
pnpm install
pnpm dev
```
